Install:
    1. git clone;
    2. run "composer install";  
    3. run "npm install";  
    4. rename .env.example to .env;
    5. configure DB and app nameg in .env config file;
    6. run "php artisan key:generate";
    7. run "php artisan jwt:secret;
    8. run "php artisan migrate.
